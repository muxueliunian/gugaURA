# gugaURA v2.0.6

## 更新内容

- 新增“工具设置”页面，可在配置工具内开启或关闭 Windows 开机自启。
- 新增应用更新检查，可读取 GitHub 最新 Release 并提示是否存在新版本。
- 发现新版本时，可从配置工具打开系统浏览器跳转到 GitHub Release 下载页面。

## 调整与修复

- 更新检查使用语义化版本比较，避免 `2.0.10` 与 `2.0.9` 这类版本被字符串顺序误判。
- 当当前版本高于 GitHub 最新 Release 时，工具设置页会提示“当前版本尚未发布”，避免发布前检查显示成“已与最新发布版本一致”。
- 开机自启入口名称固定为 `gugaURA`，便于在 Windows 启动项中识别。

## 下载

- 推荐下载：`gugaURA_installer.exe`

## 验证

- 已验证“检查更新”功能可正常读取 GitHub Release 状态。
- 已验证“开机自启”开关可正常保存状态。
